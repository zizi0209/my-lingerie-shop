 ## How to Add Screenshots
 
 To add screenshots to this documentation:
 
 1. Take screenshots of each section
 2. Save them in `documentation/static/img/screenshots/`
 3. Replace placeholder text with image markdown:
 
 ```markdown
 ![Homepage](../../../static/img/screenshots/homepage.png)
 ```
 
 4. Add captions and annotations as needed
 5. Ensure all images are optimized (max 1MB per image)
 
 ## Screenshot Guidelines
 
 - **Resolution**: 1920x1080 for desktop, 375x667 for mobile
 - **Format**: PNG for UI, JPG for photos
 - **Size**: Optimize images to < 500KB
 - **Privacy**: Blur or remove sensitive data
 - **Consistency**: Use same test data across screenshots
 - **Annotations**: Use arrows/highlights sparingly
