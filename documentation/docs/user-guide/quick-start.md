 # Quick Start
 
 Hướng dẫn nhanh để bắt đầu sử dụng My Lingerie Shop Dashboard.
 
 ## Đăng nhập
 
 1. Truy cập URL dashboard: `https://your-domain.com/dashboard`
 2. Nhập email và mật khẩu admin
 3. Click **Đăng nhập**
 
 :::info Tài khoản mặc định
 - Email: `admin@example.com`
 - Password: `admin123`
 :::
 
 ## Thêm sản phẩm mới
 
 ### Bước 1: Vào trang Products
 
 Từ sidebar, click vào **Products** → **Add New Product**
 
 ### Bước 2: Điền thông tin
 
 ```
 Tên sản phẩm: Áo ngực ren cao cấp
 SKU: BRA-001
 Giá: 299,000 VND
 Mô tả: Áo ngực ren mềm mại, thoáng mát
 ```
 
 ### Bước 3: Cấu hình Size
 
 Chọn các size có sẵn:
 - Cup: A, B, C, D
 - Band: 70, 75, 80, 85
 
 ### Bước 4: Upload hình ảnh
 
 - Kéo thả hoặc chọn file từ máy tính
 - Tối đa 5 hình ảnh
 - Định dạng: JPG, PNG, WEBP
 
 ### Bước 5: Publish
 
 Click **Save & Publish** để xuất bản sản phẩm.
 
 ## Kiểm tra Size System
 
 1. Vào trang sản phẩm vừa tạo
 2. Nhập số đo khách hàng
 3. Xem gợi ý size tự động
