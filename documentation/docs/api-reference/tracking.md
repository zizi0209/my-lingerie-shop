# Tracking API

## Endpoints

| Method | Path | Auth | Mô tả |
| --- | --- | --- | --- |
| POST | `/api/tracking/page-views` | Public | Track page view |
| GET | `/api/tracking/page-views/analytics` | Public | Analytics page view |
| POST | `/api/tracking/product-views` | Public | Track product view |
| GET | `/api/tracking/product-views/analytics` | Public | Analytics product view |
| POST | `/api/tracking/cart-events` | Public | Track cart event |
| GET | `/api/tracking/cart-events/analytics` | Public | Analytics cart event |
