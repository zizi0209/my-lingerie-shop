# Background Removal API

Tất cả endpoint yêu cầu **Admin**.

## Endpoints

| Method | Path | Auth | Mô tả |
| --- | --- | --- | --- |
| GET | `/api/background-removal/status` | Admin | Trạng thái dịch vụ |
| POST | `/api/background-removal/remove` | Admin | Xóa nền ảnh |
