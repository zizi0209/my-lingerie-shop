# Virtual Try-On API

## Endpoints

| Method | Path | Auth | Mô tả |
| --- | --- | --- | --- |

| GET | `/api/virtual-tryon/health` | Public | Health stats |
| POST | `/api/virtual-tryon/reset-health` | Public | Reset health |
