# Seed API

Các endpoint này không có bảo vệ, chỉ nên dùng trong môi trường nội bộ.

## Endpoints

| Method | Path | Auth | Mô tả |
| --- | --- | --- | --- |
| POST | `/api/seed/system-config` | Public | Seed SystemConfig |
| POST | `/api/seed/admin` | Public | Seed admin |
| POST | `/api/seed/all` | Public | Seed toàn bộ |
