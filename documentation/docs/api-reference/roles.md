# Roles API

Hiện các endpoint chưa có middleware bảo vệ.

## Endpoints

| Method | Path | Auth | Mô tả |
| --- | --- | --- | --- |
| GET | `/api/roles` | Public | Danh sách role |
| GET | `/api/roles/:id` | Public | Chi tiết role |
| POST | `/api/roles` | Public | Tạo role |
| PUT | `/api/roles/:id` | Public | Cập nhật role |
| DELETE | `/api/roles/:id` | Public | Xóa role |
