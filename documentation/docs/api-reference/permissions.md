# Permissions API

Hiện các endpoint chưa có middleware bảo vệ.

## Endpoints

| Method | Path | Auth | Mô tả |
| --- | --- | --- | --- |
| GET | `/api/permissions` | Public | Danh sách permission |
| GET | `/api/permissions/:id` | Public | Chi tiết permission |
| POST | `/api/permissions` | Public | Tạo permission |
| PUT | `/api/permissions/:id` | Public | Cập nhật permission |
| DELETE | `/api/permissions/:id` | Public | Xóa permission |
